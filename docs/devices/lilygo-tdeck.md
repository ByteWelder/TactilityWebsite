# LilyGo T-Deck

## Features

- ✅ SD card
- ✅ Power status
- ✅ USB Storage
- Keyboard

## Issues

When a battery is not present, the device sometimes fails to initialize touch. Disconnect the USB cable, wait 5 seconds and try again.
