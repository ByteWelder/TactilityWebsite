# LilyGo T-Lora Pager

## Features

- ✅ SD card
- ✅ Keyboard
- ✅ Power status
- ✅ USB Storage
- ✅ GPS (GNSS via MIA-M10Q)

## Not yet implemented

- LoRa: Semtech SX1262
- NFC: ST25R3916
- Smart sensor: Bosch BHI260AP
- Real-time clock: NXP PCF85063A
- Battery charger: Ti BQ25896
- Haptic driver: Ti DRV2605
- Audio codec: Everest-semi ES8311
- GPIO expander: XINLUDA XL9555
